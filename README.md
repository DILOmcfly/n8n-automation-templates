## How to Use

1. Import any `.json` file into your N8N instance (Settings ‚Üí Import Workflow)
2. Configure credentials (API keys, OAuth) for each service
3. Replace placeholder values (`YOUR_SHEET_ID`, `YOUR_PHONE_NUMBER_ID`, etc.)
4. Activate and test
